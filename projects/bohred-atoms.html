Bohred Atoms

  • Engine: Unity
  • Language: C#
  • Platforms: Windows PC and Web
  • Published: Feb 2017

These atoms are so bohred making things up all the time they can’t take it anymore! This time, they are going to destroy.

Bohred Atoms is a local multiplayer 2D shooter game where each player shoots the other player’s electrons with their nuclear bullets to destroy them.

This game was the first game I finished that I could call my own. Before that I had only made clones of classic games from tutorials. The game concept was based on the Bohr Model of the atom which was proposed by Danish physicist Niels Bohr. This model represents the atom as a nucleus with electrons orbiting it in circular orbits. I had learned about it in Chemistry class and thought it would be a nice idea to implement in a game.

One of the challenges I encountered was making the electrons orbit the nucleus while staying in their orbits when the atom moves. The atoms in my failed attempts literally exploded ☢. It took me a while to get right but perseverance (and Google) is key. The rest of the game mechanics were comparable to those of classic games like Breakout, Space Invaders and Pong, so I could apply my knowledge from tutorials there.

In the end I was pretty happy with how it turned out. The scope was just right for me not to be completely lost yet still learn a lot. I plan to come back to refactor the code and add an AI. But for now, onward to the next project!
