Bohred Atoms

  • Engine: Unity
  • Language: C#
  • Platforms: Windows PC and Web
  • Published: Feb 2017

These atoms are so bohred making things up all the time they can’t take it anymore! This time, they are going to destroy.

Bohred Atoms is a local multiplayer 2D shooter game where each player shoots the other player’s electrons with nuclear bullets to destroy them.

This game was the first game I finished that I could call my own. The game concept was based on the Bohr Model of the atom with game mechanics comparable to those of classic games like Pong, Breakout and Space Invaders. I had learned about the Bohr Model in Chemistry class and thought it would be fun to implement in a game.

One of the challenges I encountered was making the electrons orbit the nucleus while staying in their orbits when the atom moves. The atoms in my failed attempts literally exploded ☢. In hindsight it wasn't a difficult problem to solve, but back then it taught me a lot about how to find and come up with solutions.

In the end I was very happy with how it turned out. The scope was just right for me not to be completely lost yet still learn a lot. If I were to do it again though I would make the code more object-oriented and maybe add more weapons and an AI. But for now, onward to the next project!
