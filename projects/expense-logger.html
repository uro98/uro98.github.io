Expense Logger

  • IDE: Android Studio
  • Language: Java
  • Database: SQLite
  • Library: MPAndroidChart
  • Platform: Android
  • Published: Oct 2017

Download APK File

To install the app, you must first enable installation of third party apps on your phone. To do this, go to Settings > Security and enable Unknown sources.

Install

The first step to saving money is to see where your money is going! Expense Logger allows you to easily log your expenses wherever you go and see reports analyzing your monthly or yearly spending.

Features
  • Intuitive design: Easy to use user interface
  • Quick navigation: The calendar allows you to jump to a date to add expense records
  • Categories: Categorize your expenses and add custom categories in addition to the default ones if needed
  • Expense description: An optional note allows you to add more information about an expense
  • Interval analyses: Pie charts and bar charts let you see how much is spent where every month or year
  • Aesthetics: Visually appealing charts with beautiful colours
  • All expense list: See your most expensive expenses of all time with an ordered list
